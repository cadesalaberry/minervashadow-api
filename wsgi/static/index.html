minervashadow

Rethinking the interface of the aging minerva website.

It gathers all the most used links from minerva into one sleek and mobile enabled interface. To do so it uses a RESTful JSON API located here .

Can I have your McGill account password ? For obvious security reasons I do not want to have this website be something more than a demo yet. However if more people get interested in the project, I will certainly ask for a security audit from McGill, and/or integrate with their login system.

What technology did you use ?

The project has been entirely developed in Python 2.7 (for compatibility), using docopt for the cli, mechanize as a headless browser, beautifoulsoup4 for the html parser, and Flask as the webserver.
